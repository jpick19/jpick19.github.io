Text

No matter the size of the window, the text will adjust to it to make sure it is still readable.

Image

If for some reason the image of myself on the About Me page doesn't load, or the user uses a screen reader, it will be replaced by text.

Links

Every link for different parts of the website will be on the top right of the home page, and are labelled accurately. Any links that are clicked on this website, will be opened up in the same window. To go back to the previous page, just click on the browser's back button on the top left.
